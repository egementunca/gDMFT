# Release checklist

- [ ] Version and changelog are finalized.
- [ ] License and citation metadata are present.
- [ ] External licenses and citations are recorded.
- [ ] Dataset manifests and checksums validate.
- [ ] No machine-specific paths or credentials are tracked.
- [ ] Wheel and source distribution build in a clean environment.
- [ ] CI passes on supported Python versions.
- [ ] Public API and examples are documented.
- [ ] Registered figures rebuild from immutable dataset versions.
- [ ] Large datasets have an immutable archive and citation.
