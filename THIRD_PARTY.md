# Third-party software and data

gDMFT does not currently vendor third-party source code.

External implementations used for comparisons are recorded in
`external/sources.toml` with immutable revisions, upstream licenses, and
citations. They are installed separately and retain their own license terms.

The initial registry contains:

- `gem`, Ghost Embedding Method, GPL-3.0-or-later, pinned by commit.

Reference datasets are distributed only when their license or permission
allows it. Otherwise gDMFT records citation and retrieval metadata without
copying the underlying file.
